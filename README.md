# tl-work-sol

To install dependencies:

```bash
bun install
```

To run, go to the specific app at the `apps` directory and run:

```bash
bun dev
```

To build, go to the specific app at the `apps` directory and run:

```bash
bun run build
```

Or follow the commands in the `package.json` file.

---

This project was created using `bun init` in bun v1.2.10. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
